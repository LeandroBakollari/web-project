## Download Node.js ->
1. Go to https://nodejs.org
2. Download the LTS version
3. Install it (just keep clicking "Next")
4. Restart your terminal (or your PC if needed)

## Check if it worked ->
Open a terminal (CMD or PowerShell) and run:  
```bash
node -v  
npm -v
```
